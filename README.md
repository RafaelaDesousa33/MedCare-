# MedCare

Landing page de uma clínica médica desenvolvida com HTML e CSS.

O projeto apresenta uma interface moderna, simples e responsiva, com informações sobre a clínica, serviços médicos e formas de contato.

## Tecnologias

* HTML5
* CSS3

## Objetivo

Praticar o desenvolvimento de interfaces web modernas, responsivas e focadas na experiência do usuário.
